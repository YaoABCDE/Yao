# Debian 准备工作

::: warning

该文档有待补充和完善

:::

https://www.debian.org/doc/manuals/debian-faq/getting-debian.zh-cn.html

```bash
#修改登录账户密码
passwd

# 查看 Shell
cat /etc/shells
echo $SHELL

#改为使用 bash
chsh -s /bin/bash

# 查看linux 版本
lsb_release -a

# 修改  hostname  添加
vim /etc/hostname
vim /etc/hosts


```

## 修改 ssh 配置

vim /etc/ssh/sshd_config

```YML

PermitRootLogin yes

StrictModes no

PubkeyAuthentication yes

PasswordAuthentication yes

UsePAM yes

TCPKeepAlive yes

ClientAliveInterval 60

ClientAliveCountMax 3

```

然后重启 ssh

```bash
service ssh restart
```

## 保持系统最新

```bash
sudo apt update
sudo apt upgrade

```
