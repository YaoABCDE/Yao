# 5.远程开发

## Remote Development

远程开发工具包包含四个插件，
加上 `Remote Explorer` 一共五个插件。

让你可以通过 VSCode 远程连接服务器进行开发作业，如果网络延迟足够低，感觉就跟本地开发没啥区别，体验丝滑。
包括 WSL 支持。

https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack
